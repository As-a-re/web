AdminHack

Hacking an admin panel refers to gaining unauthorized access to the administrative control panel of a website or

web application. This can be achieved through various techniques such as SQL injection, cross-site scripting,

brute-force attacks, and other methods.

Installation

  • pkg upgrade
  • pkg update
  • pkg install git
  • git clone https://github.com/mishakorzik/AdminHack
  • cd AdminHack
  • bash setup.sh
